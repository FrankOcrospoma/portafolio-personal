Detalle Comprobante

Métodos para registrar los detalle de comprobantes que adquirio el huesped en el hotel.

Obtener todos los detalles de los comprobantes

Este método te permite obtener todos los detalles de los comprobantes registrados en el hotel.

HTTP Request (GET) https://grupo3.pythonanywhere.com/api_obtenerdetalles
No es necesario preparar un cuerpo en la solicitud
RESPONSE
[
{
"comprobante_id": 2,
"monto": "10.00",
"servicios_id": 1
},
{ "comprobante_id": 1,
"monto": "50.00",
"servicios_id": 2
},
{
"comprobante_id": 2,
"monto": "500.00",
"servicios_id": 3
}, {
"comprobante_id": 2,
"monto": "510.00",
"servicios_id": 4
},
{ "comprobante_id": 1,
"monto": "240.00",
"servicios_id": 6
}
]		 El cuerpo de respuesta, manifiesta todos los elementos pedidos con siete campos, a través de un JSON array.
Donde:
comprobante_id: Es la foranea de comprobante
monto: Es el total de dinero que registra
servicios_id: Es la foranea de servicio

Guardar detalle comprobante

Este método te permite crear un detalle comprobante nuevo.

HTTP Request (POST) https://grupo3.pythonanywhere.com/api_guardardetalle
{ {
"comprobante_id": 4,
"monto": "150.00",
"servicios_id": 2
}
}		 El cuerpo de solicitud, manifiesta todos los elementos a pedir con seis campos, a través de un JSON array. Aqui no se coloca el campo id, el sistema se encargará de asignarle el id correspondiente.
Donde:
comprobante_id: Es la foranea de comprobante
monto: Es el total de dinero que registra
servicios_id: Es la foranea de servicio
RESPONSE
{ "Mensaje": "Detalle Comprobante registrado correctamente" } El cuerpo de respuesta, manifiesta que el detalle comprobante ha sido registrado correctamente. Donde Mensaje: Es la respuesta de la lógica de negocio de detalle comprobante.
