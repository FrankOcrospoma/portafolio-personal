Categoría Habitacion

Métodos para registrar las Categorías Habitaciones que tiene el hotel para el huesped.

Obtener todas las Categoría Habitaciones

Este método te permite obtener todas las Categoría Habitacion registradas por el hotel.

HTTP Request (GET) https://grupo3.pythonanywhere.com/api_obtenercategoria_habitacion
No es necesario preparar un cuerpo en la solicitud
RESPONSE
[
{
{ "id": 3,
"nombre": "Matrimonial",
"precio": "150.00" }br> "id": 2 },
{
{ "id": 4,
"nombre": "Suite",
"precio": "250.00" } }
]		 El cuerpo de respuesta, manifiesta todos los elementos pedidos con dos campos, a través de un JSON array.
Donde
nombre: Es el nombre de la categoría de habitación a elegir por el huesped.
precio: Es el precio la categoría habitación a ofrecer al huesped.
id: Es la identificación de la Categoría Habitacion.

Guardar Categoría Habitación

Este método te permite crear una nueva habitación.

HTTP Request (POST) https://grupo3.pythonanywhere.com/api_guardarcategoria_habitacion
{ "id": 7,
"nombre": "Triple",
"precio": "300" }		 El cuerpo de solicitud, manifiesta todos los elementos a pedir con un campos, a través de un JSON array. Aqui no se coloca el campo id, el sistema se encargará de asignarle el id correspondiente.
Donde
nombre: Es el nombre de la categoría de habitación a elegir por el huesped.
precio: Es el precio la categoría habitación a ofrecer al huesped.
id: Es la identificación de la Categoría Habitacion.
RESPONSE
{ "Mensaje": "Categoría Habitacion registrada correctamente" } El cuerpo de respuesta, manifiesta que la Categoría Habitacion ha sido registrada correctamente. Donde Mensaje: Es la respuesta de la lógica de negocio de Categoría Habitacion.
