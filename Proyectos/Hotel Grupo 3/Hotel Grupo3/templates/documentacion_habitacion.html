Habitación

Métodos para registrar las habitaciones que tiene el hotel para el huesped.

Obtener todas las habitaciones

Este método te permite obtener todas las habitaciones registradas por el hotel.

HTTP Request (GET) https://grupo3.pythonanywhere.com/api_obtener_habitaciones
No es necesario preparar un cuerpo en la solicitud
RESPONSE
[
{
"categoria": "Matrimonial",
"descripcion": "vista al mar",
"estado_habitacion": "Disponible",
"id": 2 },
{
"categoria": "Doble",
"descripcion": "Con balcon",
"estado_habitacion": "Disponible",
"id": 6 }
]		 El cuerpo de respuesta, manifiesta todos los elementos pedidos con dos campos, a través de un JSON array.
Donde
categoria: Es el tipo de habitación a elegir por el huesped.
descripcion: Es la breve explicación escrita de la habitación a ofrecer al huesped, considerando su aspecto importante.
id: Es la identificación de la habitación.

Guardar habitación

Este método te permite crear una nueva habitación.

HTTP Request (POST) https://grupo3.pythonanywhere.com/api_guardarhabitacion
{ "categoria": 5,
"descripcion": "Netflix + tv",
"estado_habitacion": "D" }		 El cuerpo de solicitud, manifiesta todos los elementos a pedir con un campos, a través de un JSON array. Aqui no se coloca el campo id, el sistema se encargará de asignarle el id correspondiente.
Donde
id: Es la identificación de la habitación.
descripcion: Es la breve explicación escrita de la habitación a ofrecer al huesped.
estado_habitacion: Es el estado actual de la habitación a elegir por el huesped.
categoria: Es el id del tipo de habitación a elegir por el huesped.
RESPONSE
{ "Mensaje": "Habitación registrada correctamente" } El cuerpo de respuesta, manifiesta que la habitación ha sido registrada correctamente. Donde Mensaje: Es la respuesta de la lógica de negocio de habitación.
