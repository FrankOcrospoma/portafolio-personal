Detalle Alojamiento

Métodos para registrar los detalles de alojamiento que tiene el hotel para el huesped.

Obtener todos los detalles de alojamiento

Este método te permite obtener todos los detalles de alojamiento registrados por el hotel.

HTTP Request (GET) https://grupo3.pythonanywhere.com/api_obtenerdetalle_alojamientos
No es necesario preparar un cuerpo en la solicitud
RESPONSE
[
{
"id": 2,
"persona_id": 8,
"transaccion_id": 3
},
{
"id": 3,
"persona_id": 8,
"transaccion_id": 1
}
]		 El cuerpo de respuesta, manifiesta todos los elementos pedidos con tres campos, a través de un JSON array.
Donde
id: Es la identificación del detalle de alojamiento.
persona_id: Es la identificación de la persona.
transaccion_id: Es la identificación de la transaccion.

Guardar detalle alojamiento

Este método te permite crear un detalle de alojamiento nuevo.

HTTP Request (POST) https://grupo3.pythonanywhere.com/api_guardardetalle_alojamiento
{ "persona_id": 4,
"transaccion_id": 3
}		 El cuerpo de solicitud, manifiesta todos los elementos a pedir con dos campos, a través de un JSON array. Aqui no se coloca el campo id, el sistema se encargará de asignarle el id correspondiente.
Donde
persona_id: Es la identificación de la persona.
transaccion_id: Es la identificación de la transaccion.
RESPONSE
{ "Mensaje": "Detalle alojamiento guardada correctamente" } El cuerpo de respuesta, manifiesta que el detalle alojamiento ha sido guardada correctamente. Donde Mensaje: Es la respuesta de la lógica de negocio de detalle alojamiento.
